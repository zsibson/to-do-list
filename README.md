# to-do-list

A simple to do list web application with the purpose of getting things done. 
Built with node.js, express.js, mongodb, and ejs. 

## Install Dependencies
`npm init -y`
`npm i nodemon`
`npm i express`
`npm i mongoose`
`npm i dotenv`
